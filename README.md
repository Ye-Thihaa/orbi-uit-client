## 🎯 **Quick Start**

```bash
# 🚀 Clone the universe
git clone https://github.com/your-username/orbiuit.git

# 📦 Install dependencies
cd orbiuit
npm install

# 🔥 Launch into orbit
npm start

# 🌟 Access at localhost:3000
```
